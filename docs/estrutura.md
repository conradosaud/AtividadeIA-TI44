# Estrutura
- Use **Orientação a Objetos** com:
    - Encapsulamento
    - Getters e Setters
    - Construtores
- Toda classe deve ter a função `Mostrar()` para exibir suas informações
- Use o padrão **Active Records**

## Depedências
Este projeto usa o **MySQL Connector** para conectar ao banco de dados
